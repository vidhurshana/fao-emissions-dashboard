# 🌱 Agrifood GHG Emissions Dashboard

Interactive dashboard exploring greenhouse gas emissions from agrifood systems across 234 countries (1961–2021).

## Dataset
FAO Emissions Totals via World Bank Data360

## Setup
```bash
pip install -r requirements.txt
streamlit run App.py
```
